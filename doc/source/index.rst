SPAGAT - A Spatial AGgregation Analysis Tool in Python
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

The SPAGAT framework provides the tools to analyze the impact of spatially aggregating (energy system) data.

**Features**

* application of different spatial clustering algorithms to reduce spatial complexity of data
* separation between grouping (reducing the number of regions) and representation (representing information for the reduced number of regions)
